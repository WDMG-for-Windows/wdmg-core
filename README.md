1. Download latest release
2. Open file explorer
3. Navigate to .zip
4. Select it and click Extract All
5. **Do not touch wdmg.ico**
6. Run install.ps1 in the extracted folder and click Yes on UAC prompt.
7. Select what you want and click Install.
8. Wait for it to complete (usually a few seconds)
9. Aceept the restart popup or restart later if needed.
10. Once your system reboots, it now supports .wdmg files
11. **Do not delete the folder, it is very important for installing and uninstalling.**
# Uninstallation

1. Go to the folder
2. Click uninstall.ps1
3. Agree to any UAC prompts if any show up
4. Once you've chosen whether to keep the icon or not, click the button to start uninstallation
5. Explorer will restart, and you've successfully uninstalled WDMG.

**To download any prebuilt .wdmg files, [visit the WDMG Library](https://github.com/WDMG-for-Windows/wdmg-library)**
